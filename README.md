# WiredPeers

> **One platform. Every peer. Total flow.**

A modern, stylish landing page for WiredPeers — the peer-to-peer collaboration platform.

## Deployment

This site is designed to be deployed as a GitHub Pages site. Simply enable GitHub Pages in the repository settings and point it to the `main` branch root.

## Stack

- Pure HTML & CSS (no build step required)
- Google Fonts (Inter)
- Responsive design
- CSS animations and gradients
